# Auto Installer Termux

One-command Google Drive downloader and installer for Termux.

## Quick Install

```bash
# Single download
curl -sSL https://raw.githubusercontent.com/username/auto-installer/main/install.sh | bash

# With link directly
curl -sSL https://raw.githubusercontent.com/username/auto-installer/main/install.sh | bash -s -- "YOUR_GOOGLE_DRIVE_LINK"

# Multi download
curl -sSL https://raw.githubusercontent.com/username/auto-installer/main/multi_install.sh | bash

# Batch install from file
curl -sSL https://raw.githubusercontent.com/username/auto-installer/main/batch_install.sh | bash
